It integrates with Home Assistant via a FastAPI proxy with per-agent access control. Agent personas are configured via SOUL.md and IDENTITY.md files. ## OpenBrain / PostClaw — Persistent Memory Layer

OpenBrain is the persistent memory layer for OpenClaw, implemented as PostClaw — a PostgreSQL plugin using pgvector and FastAPI MCP. It replaces the original flat-file memory system with semantic vector search, separating episodic (short-term conversation) and semantic (long-term factual) memory with automated maintenance via a sleep cycle.
